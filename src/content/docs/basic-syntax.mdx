## Status

Basic syntax is stable in the interpreter. The JIT lowers a useful subset and skips unsupported constructs cleanly.

## File Shape

A Cx file is a list of declarations and top-level statements. A `main` function is optional for simple scripts. Top-level statements run.

<SourceComment source="origin/submain:examples/hello.cx" />
```cx
name: str = "World"
age: t32 = 42
score: f64 = 9.8

print("Hello, {name}!")
print("Age: {age}")
print("Score: {score}")
```

## Comments

Line comments and block comments are accepted.

<SourceComment source="origin/submain:src/tests/verification_matrix/t76_block_comments.cx" />
```cx
/# this is a block comment #/
x: t32 = 10
/# inline block comment #/
/#
   multi
   line
   comment
#/
print(x)
```

## Variables

Types follow the name after `:`. Untyped `let` declarations infer from later assignment.

<SourceComment source="origin/submain:src/tests/verification_matrix/t122_vardecl_int_exit.cx" />
```cx
x: t64 = 42
assert_eq(x, 42)
y: t64 = 100
assert_eq(y, 100)
```

<SourceComment source="origin/submain:src/tests/verification_matrix/t17_unknown_literal.cx" />
```cx
let x;
x = ?
print(x)
```

## Functions

Functions use `fnc:`. A return type appears between `fnc:` and the function name. If there is no return value, omit the type.

<SourceComment source="origin/submain:examples/fibonacci.cx" />
```cx
fnc: t64 fib(n: t64) {
    if n < 2 {
        return n
    }
    a: t64 = 0
    b: t64 = 1
    tmp: t64 = 0
    i: t64 = 2
    while (i <= n) {
        tmp = a + b
        a = b
        b = tmp
        i += 1
    }
    return b
}
```

## Operators

Arithmetic: `+`, `-`, `*`, `/`, `%`.

Comparison: `==`, `!=`, `>`, `<`, `>=`, `<=`.

Logical: `&&`, `||`, `!`.

Compound assignment uses standard infix form.

<SourceComment source="origin/submain:src/tests/verification_matrix/t151_var_compound_assign_exit.cx" />
```cx
x: t64 = 10
x += 5
assert_eq(x, 15)

x = 17
x %= 5
assert_eq(x, 2)
```

Use `%=` for compound modulo assignment.

## Macros

Outer macros use `#[name]`. The parser accepts a closed set. Unknown names are rejected.

<SourceComment source="origin/submain:src/tests/verification_matrix/t67_macro_outer_test.cx" />
```cx
#[test]
fnc: run_test() {
    print(1)
}

run_test()
```
