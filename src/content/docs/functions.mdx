## Status

Functions are stable in the interpreter. The JIT lowers direct calls, arity checks, type checks, return values, void returns, and exit-code propagation. DirectCall still has SKIP fixtures.

## Definition

Return types go after `fnc:` and before the function name.

<SourceComment source="origin/submain:examples/fibonacci.cx" />
```cx
fnc: t64 fib(n: t64) {
    if n < 2 {
        return n
    }
    a: t64 = 0
    b: t64 = 1
    tmp: t64 = 0
    i: t64 = 2
    while (i <= n) {
        tmp = a + b
        a = b
        b = tmp
        i += 1
    }
    return b
}
```

## Implicit Return

The final bare expression can return a value.

<SourceComment source="origin/submain:src/tests/verification_matrix/t02_implicit_return.cx" />
```cx
fnc: t64 add(a: t64, b: t64) {
    a + b
}
print(add(10, 20));
```

## Explicit Return

Use `return` when early exit is clearer.

<SourceComment source="origin/submain:src/tests/verification_matrix/t03_explicit_return.cx" />
```cx
fnc: t64 find(a: t64, b: t64) {
    return a;
    b
}
print(find(10, 20));
```

## Forward Calls

Top-level forward calls work in the interpreter and have partial JIT coverage.

<SourceComment source="origin/submain:src/tests/verification_matrix/t29_forward_decl.cx" />
```cx
outer()

fnc: outer() {
    print(99)
}
```

## Nested Functions

Nested functions are scoped inside the containing function.

<SourceComment source="origin/submain:src/tests/verification_matrix/t14_nested_5_deep.cx" />
```cx
fnc: t64 level1() {
    fnc: t64 level2() {
        fnc: t64 level3() {
            fnc: t64 level4() {
                fnc: t64 level5() {
                    5
                }
                level5() + 1
            }
            level4() + 1
        }
        level3() + 1
    }
    level2() + 1
}
print(level1());
```

## Copy Parameters

`.copy`, `.copy.free`, and `copy_into(...)` work in the interpreter. Treat the long-term semantics as still in progress.

<SourceComment source="origin/submain:src/tests/verification_matrix/t24_full_system_regression.cx" />
```cx
x: t64 = 10
fnc: t64 bump(n.copy) {
    n = n + 5
    n
}
print(bump(x.copy))
print(x)
```

<SourceComment source="origin/submain:src/tests/verification_matrix/t24_full_system_regression.cx" />
```cx
a: t64 = 10
b: t64 = 20
c: t64 = 30
fnc: t64 sum_box(t: copy_into(a, b, c)) {
    t.a + t.b + t.c
}
print(sum_box(copy_into(a, b, c)))
```
